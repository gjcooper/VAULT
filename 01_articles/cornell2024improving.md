## My Thoughts


This paper details the testing of a model of cued memory recall. It first details a free-recall memory model (the [[Context Maintenance and Retrieval]] model), then extends it for cued memory recall by assuming that the context for items after a cue is updated one time to the exact context of the cued item.

This relies on the fact that the cue is an unrecalled list item. I'm not entirely sure how this would be extended to use semantically related, but unstudied words.

One interesting approach here is to fit the model to pilot data, and then fix the model parameters. In the main dataset these fixed model parameters are used (and give impressive fits to data). Post-cue the expected effects are realised in that if a cue is selected based on semantic similarity (the words on the list were drawn from four categories) combined with temporal similarity (close to missing items on the list) then posdt-cue recall improves.

In fact the post-cue recall showed an effect where recall was better for best cue > random cue > worst cue.


#### How do they know this?

See above for their approach. One thing that slightly detracts from this is the fact that all mode fits are done to aggregate data. This actually is a mixed bag - it strengthens the pre-registered approach and general applicability of the model. However it also fuzzes the results for individuals.

#### What would I need to believe to accept this argument?

Extensions into individual data and/or individual subject predictions based off of the model in its current form.

#### What questions does this raise that aren’t addressed?

What would happen with cues that are semantically related but not on the studies list? How would this effect the model structure?
