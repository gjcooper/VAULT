## My Thoughts

Wanted to give this a read because it would be the approach I would take for the [[Test-retest reliability of psychiatric battery tests without response styles]] project idea that I have.

Whilst I have heard a lot about this project before from [[Jess Grimmond]] and [[Guy E. Hawkins]] I do have some notes for things I was not aware of.

#### What it is good for

Separating out the underlying latent state that a survey/questionnaire is attempting to get information about. The example application from the paper is the [[International personality Item Pool]] (https://openpsychometrics.org/_rawdata/) which holds responses from a large number  (1 million+) participants on a Big 5 personality survey.

The modelling technique focusing on estimating both the underlying latent state in a continuous range, as well as the response style of each participant. These response styles are easily named ones like extreme or midpoint responders, but also could encapsulate other response patterns that are not as easy to reason about.

#### Caveats

- The model needs 2 or more latent states or multiple surveys capturing different aspects, as long as all the surveys have the same response option anchoring.
- The model also requires a certain number of reverse-scored items (or it needs to drop the left/right response bias detection)
