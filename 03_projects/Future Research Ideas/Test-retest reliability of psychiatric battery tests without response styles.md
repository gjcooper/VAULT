## project idea

Using Jess's work on separating out response style from true distributions [[grimmond2025solution]], does this change the [[test-retest reliability]] of typical psychiatric battery tests like depression scales etc. These typically have high reliability compared to behavioural tasks, important for individual differences research, but works this call it into question.


# What I would need to do

- Identify a source of data that was (ideally) previously used to identify test-retest reliability of one or more psychiatric survey tests.
- Find out what ind of analysis is typically done to measure test-retest reliability
- run the standard test-retest reliability
- run test-retest reliability _after_ extracting the underlying latent state the test is supposed to measure
- Compare and contrast reliability.
- Compare and contrast to cognitive model reliability.
